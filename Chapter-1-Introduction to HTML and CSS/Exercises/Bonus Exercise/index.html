The Basic HTML and CSS Quick Reference Guide

Head & Body Sections

The head section contains information about the web page document. 

<head>
    <title>The title of the page goes here</title>
    <meta charset="utf-8">
</head>

Text and other elements that appear in the web page document are contained in the body section.

Input: 
<body>
    Here are additional HTML5 tags and body text.
</body>
Output:

Here are additional HTML5 tags and body text.

Phrase Elements

Indicate the context and meaning of the text.

Code Example Description 
<b>
 Bold text Text that has no extra importance but is styled in bold font by usage and convention. 
<em>
 Emphasized text Causes text to be emphasized in relation to other text; usually displayed in italics. 
<i>
 Italicized text Text that has no extra importance but is styled in italics by usage and convention. 
<mark>
 Marked text Text that is highlighted in order to be easily referenced. 
<small>
 Small text Legal disclaimers and notices (“fine print”) displayed in small font-size. 
<strong>
 Strong text Strong importance; causes text to stand out from surrounding text; usually displayed in bold. 
<sub>
 H2O Subscript displays a subscript as small text below the baseline. 
<sup>
 x2 Superscript displays a superscript as small text above the baseline.

HTML Lists

Unordered List

An unordered list displays a bullet before each entry.

Input: 
<p>
    An unordered list of network protocols:
</p>

<ul>
    <li>TCP</li>
    <li>IP</li>
    <li>HTTP</li>
    <li>FTP</li>
</ul>
Output:

An unordered list of network protocols:

  • TCP
  • IP
  • HTTP
  • FTP

An ordered list displays numbers or letters before each entry.

    <li>Apply to school</li>
    <li>Register for course</li>

  1. Apply to school
  2. Register for course
  3. Pay tuition
  4. Attend course

Div Element

The <div> element configures a structural block area or "division" on a web page with empty space above and below. You can use CSS to style and position the divs by adding class and/or id attributes to your div elements. This is the foundation for CSS-based page layouts.

Input: 
<div class="description">
    <h1>This is the first heading</h1>
    <p>Description Here</p>
</div>
Output:

This is the first heading

Description Here

Span Element

The <span> is an inline element that is often used as a container for some text. It is used to style parts of the text.

The span element is similar to div; the difference is that div is a block-level element, whereas span is an inline element.

Block-level Elements: Always start on a new line. Intended for holding standalone blocks of content, such as paragraphs of text.

Inline Elements: Intended to hold smaller pieces of content within a larger block of content, such as a few words or a sentence.

Input: 
<h2>Some <span style="color:red">Important</span> Message</h2>
Output:

This is an important message!

CSS: Cascading Style Sheets

CSS (Cascading Style Sheets) allows you to define styles for web pages. It targets elements using selectors, which allows you to target specific elements and apply the same style to them.

CSS STRUCTURE (syntax)

Selector {
    Property 1: value;
    Property 2: value;
    Property 3: value;
}

Applying CSS into HTML

Inline CSS

To apply a distinct style to a single HTML element. Inline CSS makes use of an HTML element's style attribute.

Input: 
<h1 style="color: White;">
    Hello World!
</h1>
Output:

Hello World!

Internal CSS

To define a distinct style to a single HTML page. Internal CSS is defined in the <head> section of an HTML page, within a <style> element.

Input: 
<head>
    <h1>
        Hello!
    </h1>
    <style>
        h1 {
            color: Red;
        }
    </style>
</head>
Output:

Hello!

External CSS

To apply a single distinct style to multiple HTML pages. External CSS is defined in a separate file with a .css extension, and linked to an HTML page using the <link> element within the <head> section.

<head>
    <link rel="stylesheet" type="text/css" href="style.css">
</head>

Box in CSS

Each HTML element is presented as a rectangular window by a browser. Afert that CSS is added to the box to style HTML element

The box section in CSS refers to the box layout of each HTML element.
Each box consists of the following:

  1. Content:

    actual cotent (text, images, video, etc) of the HTML element.
  2. Padding:

    Define by the padding Property. In-between the border and content of an elements. Is transpant.
  3. Border:

    Defined by the border property. Surrounds the padding and content of an element.
  4. Margin:

    Defined by the margin property. Adds space between the border and other HTML elements. Is transparent.

code {
    background-color: #White;
    border: 3px dashed #000000;
    color: #000000;
    display: black;
    padding-left: 50px;
    padding-bottom: 5px;
    font-size: 30%;
    }
